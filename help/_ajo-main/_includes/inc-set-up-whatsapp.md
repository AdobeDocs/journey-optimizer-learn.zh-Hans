# 设置 WhatsApp 频道

本教程将指导您在 Adobe Journey Optimizer 中设置 WhatsApp 渠道以启用实时业务消息。

您将学习的内容：

* 添加并配置 WhatsApp API 凭据和业务帐户详细信息。
* 设置 Webhook 以进行入站消息处理，包括选择启用/选择禁用关键词。
* 创建渠道配置以完成 WhatsApp 设置。

非常适合希望在客户历程中激活 WhatsApp 消息的营销人员和开发人员。
