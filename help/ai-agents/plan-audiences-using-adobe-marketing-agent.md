# 使用 Adobe Marketing Agent for Microsoft 365 Copilot 规划受众

了解 Adobe Marketing Agent for Microsoft 365 Copilot 如何直接在 Microsoft Teams 中简化受众规划。借助此工具，营销人员无需切换平台或向运营团队提交请求，即可识别忠诚度受众、查看历程谱系以及验证营销活动候选者。

此视频演示了选择、分析和准备营销活动受众的过程。它展示了代理如何生成切实可行的洞察、显示相关建议，并在受众和历程之间提供谱系可视化。营销人员可以高效地管理数据和简化工作流，确保有重点的定位和改善营销活动结果。
