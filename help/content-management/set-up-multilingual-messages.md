# 设置多语言消息

{{limited-availability}}

Adobe Journey Optimizer多语言消息功能允许您在单个营销活动或历程中轻松创建多种语言的内容。 在编辑活动或历程时，您可以在语言之间切换，简化整个编辑过程，并提高有效管理多语言内容的能力。

了解如何设置自动翻译的翻译项目以及手动翻译和多语言消息的自动翻译项目所需的语言设置。
