# 报告和监测

了解 Adobe Journey Optimizer 报告如何帮助您轻松跟踪、分析和处理客户历程。

这些教程可帮助营销人员掌握各种工具，用于实时监控性能、深入挖掘详细的历程和渠道洞察、导出数据、设置警报并利用高级分析做出更明智的决策。
