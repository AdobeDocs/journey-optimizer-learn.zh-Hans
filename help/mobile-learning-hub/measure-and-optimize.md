# 衡量性能和优化

访问报表、分析绩效并优化策略以获得更好的结果：

1. 跟踪关键量度：投放次数、打开次数、点击次数和转化次数。
2. 分析受众表现以了解参与模式。
3. 根据数据见解优化消息发送时间、内容和定位。
