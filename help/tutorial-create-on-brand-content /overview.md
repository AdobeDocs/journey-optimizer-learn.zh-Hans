# 在Adobe Journey Optimizer中为营销人员创建AI支持的品牌内内容

了解如何使用Adobe Journey Optimizer的AI功能创建、优化和管理品牌内内容，从品牌设置和完整营销活动生成到视觉增强、多语言版本和自动质量检查。
专为希望跨每个渠道扩展个性化、一致内容的营销人员构建。

## 您学到的内容

* 构建和配置一个品牌配置文件，用于在Adobe Journey Optimizer、GenStudio和Adobe Express中引导人工智能生成的内容。
* 使用人工智能生成并编辑全渠道电子邮件、推送和应用程序内内容。
* 使用集成的人工智能工具增强可视化并创建多语言版本。
* 在内容上线之前，运行自动化质量检查、评估品牌一致性并应用治理。

### 简介和品牌设置

本模块将指导您逐步在Adobe Journey Optimizer中设置品牌，以使您的AI生成内容反映您的身份。 您会看到AI生成的电子邮件内容的完整示例，然后通过上传准则并映射关键元素（如语气、法律约束和图像规则）来创建和配置品牌个人资料。 您还将了解如何将该品牌扩展到GenStudio和Adobe Express中，以及如何为不同团队管理多个品牌。 最终，您拥有有效的品牌配置文件，无论您创建什么内容，它都会有效地指导您使用AI生成的内容。

* [使用AI创建品牌内电子邮件内容](./introduction.md)
了解Adobe Journey Optimizer中的AI功能如何帮助您从单个提示到完全生成的品牌内电子邮件大规模构建品牌内电子邮件内容。
* [在Adobe Journey Optimizer中为AI支持的内容配置品牌](./configure-brands.md)
创建和配置AI功能用于生成和验证内容、对应语音音调、法律约束、图像规则等的品牌。
* [跨创意工具扩展品牌指南](./brands-tips-and-tricks.md)
了解如何跨GenStudio和Adobe Express应用在Adobe Journey Optimizer中配置的品牌、添加示例以及管理多个品牌。

### 使用人工智能生成内容

本模块将向您展示如何使用Adobe Journey Optimizer中的AI功能端到端地生成和优化品牌上内容。 您可以从提示生成完整的电子邮件、推送和应用程序内资产，编辑单个部分（如主题行、预标题和图像），增强视觉效果和创建多语言版本，同时保留您的品牌声音。 到最后，你将从一个想法变成精良、多语言、品牌文案和视觉效果。

* [从头开始创建全渠道内容](./create-content-from-scratch.md)
从模板开始，使用AI功能、品牌指南以及可配置的色调和图像设置生成完整的电子邮件、推送通知或应用程序内消息。
* [编辑和个性化部分内容](./edit-and-personalize-section-content.md)
利用AI功能编辑电子邮件模板的单个部分并使其个性化，更改主题行、标头、文本块和图像，同时保留品牌声音和合规性。
* [使用AI增强您的视觉内容](./enhance-your-visual-content.md)
使用AI工具生成信息图形、添加叠加并优化图像，然后在Adobe Express中扩展编辑以生成可供重用的品牌上可视化图表。
* [创建和优化内容的多语言版本](./create-and-refine-multilingual-content.md)
使用AI功能、品牌准则和自动评分将内容翻译并调整为其他语言，使本地化内容保持品牌化。

### 质量、品牌协调和治理

此模块可帮助您在AI生成的内容上线之前对其进行评估和管理。 您可以使用品牌一致性得分根据您的指南检查内容，运行自动质量检查以查看可读性和合规性，并在审批过程中应用人工审查。 最终，您知道如何“信任但验证”AI输出，并保持每项资产合规且一致。

* [确保内容质量、品牌协调和治理](./quality-brand-alignment-and-governance.md)
评估人工智能生成和手动编写的资产的内容质量和品牌一致性，并在内容上线之前将治理作为审核流程的一部分应用。
