# 在Adobe Journey Optimizer中为营销人员创建AI支持的品牌内内容

了解如何使用Adobe Journey Optimizer的AI功能创建、优化和管理品牌内内容，从品牌设置和完整营销活动生成到视觉增强、多语言版本和自动质量检查。专为希望跨每个渠道扩展个性化、一致内容的营销人员构建。

## 您学到的内容

### 简介和品牌设置

本模块将指导您逐步在Adobe Journey Optimizer中设置品牌，以使您的AI生成内容反映您的身份。

* [简介](./introduction.md)
此简介介绍AI支持的工具如何帮助您生成电子邮件内容并应用品牌指南。
* [配置品牌](./configure-brands.md)
了解如何在Adobe Journey Optimizer中上传品牌指南、定义样式并确保跨渠道进行一致的品牌内通信。
* [品牌指南提示](./brands-tips-and-tricks.md)
了解跨Adobe Journey Optimizer、GenStudio和Adobe Express扩展品牌准则的提示，以实现更好的品牌一致性和团队协作。

### 使用人工智能生成内容

本模块将向您展示如何使用Adobe Journey Optimizer中的AI功能端到端地生成和优化品牌上内容。

* [从头开始创建内容](./create-content-from-scratch.md)
从模板开始，应用品牌指南，并使用简单的提示从头开始创建电子邮件、推送通知和应用程序内消息。
* [编辑和个性化部分内容](./edit-and-personalize-section-content.md)
了解如何使用AI在Adobe Journey Optimizer中编辑和个性化各个内容组件。
* [增强您的视觉内容](./enhance-your-visual-content.md)
了解如何在Adobe Journey Optimizer中增强可视化内容。
* [创建和优化内容的多语言版本](./create-and-refine-multilingual-content.md)
了解AI如何帮助翻译和本地化内容，同时保持品牌一致性。 了解高效的工作流以实现全局通信扩展。

### 质量、品牌协调和治理

此模块可帮助您在AI生成的内容上线之前对其进行评估和管理。
