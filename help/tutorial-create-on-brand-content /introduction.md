# 使用AI创建品牌上电子邮件内容

了解Adobe Journey Optimizer中的AI功能如何帮助您大规模构建品牌内电子邮件内容。 本演练展示了从单个提示生成的完整电子邮件，其中使用品牌准则和引用内容来指导生成、选择生成模型并将AI生成的文本和图像直接插入模板中。 相同的品牌资产还支持自动质量检查，对每个变体进行合规性、色调和品牌一致性评分。
