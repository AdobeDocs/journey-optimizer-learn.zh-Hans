# 历程画布中的内容决策活动

了解历程画布中的内容决策活动如何允许用户利用Adobe Journey Optimizer Decisioning提供个性化的内容选件。 使用此功能根据检索到的产品建议路由轮廓或将产品建议发送到外部系统，从而提高历程的有效性和个性化水平。
