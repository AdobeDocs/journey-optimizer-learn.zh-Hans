# 创建查找数据集

数据集查找功能允许 Adobe Journey Optimizer 在历程运行时从 Adobe Experience Platform 数据集中检索引用数据或交易数据，而无需将这些数据存储在实时客户轮廓中。

在此用例中，当用户提交投资偏好表单（股票、债券或定期存款）后，历程会从参考数据集中动态查询对应的投资详情，并利用该信息实现邮件内容个性化。
