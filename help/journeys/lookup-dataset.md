# 创建查询数据集

数据集查找允许Adobe Journey Optimizer在历程运行时从Adobe Experience Platform数据集检索引用或事务性数据，而无需将数据存储在实时客户个人资料上。

在此使用案例中，当用户提交投资偏好设置表单（股票、债券或CD）时，历程将从引用数据集中动态查找对应的投资详细信息，并使用该信息来个性化电子邮件。
