# 在表单提交时触发历程

此用例演示了客户表单提交如何实时触发Adobe Journey Optimizer历程，从而根据用户提供的信息实现即时、个性化的跟进通信。


## 本教程的先决条件

* 对Experience Platform的访问权限

* 基本了解Experience Platform概念（用户档案、受众、数据集）

* 熟悉Journey Optimizer

## 实施步骤

* 在Adobe Experience Platform中定义事件架构
* 在Adobe Journey Optimizer中配置自定义事件
* 配置电子邮件渠道
* 构建由自定义事件触发的历程
* 将表单提交数据发送到Adobe Experience Platform
* 配置数据集查找以在电子邮件中包含引用数据
