# Journey Optimizer 视频和教程

![](./assets/ajo-banner.png)

Journey Optimizer 帮助企业为其客户提供贴合心意的、情境式和个性化的体验。客户历程是客户与品牌互动的整个过程，从接触的第一刻起直到客户离开。

本用户指南包含有关 Adobe Journey Optimizer 特性和功能的视频和教程。
