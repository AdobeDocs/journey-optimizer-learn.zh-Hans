# 创建个性化优惠

了解如何在[!DNL Offer Decisioning]中创建个性化优惠。 个性化选件具有与其关联的资格规则，可帮助您仅向相关客户显示这些选件。 例如，您可以将雪铲的选件限制在雪下的地区，并且只能在冬季时间。
