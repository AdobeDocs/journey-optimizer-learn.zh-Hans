# 创建收藏集

了解如何在[!DNL Offer Decisioning]中创建集合。 收藏集用于管理逻辑组中的选件，并且是构建[!DNL Offer Decisioning]活动所必需的。 动态收藏集允许您根据标记对选件进行分组，而静态收藏集允许您手动组织选件。
