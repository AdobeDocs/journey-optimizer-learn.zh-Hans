# 创建规则

了解如何在 [!DNL Offer Decisioning] 中创建规则。规则或“决策规则”是个性化优惠的必需构建基块组件之一。规则是使用 Experience Platform 的 Real-time Customer Profile 中的事件和属性构建的，并基于优惠的资格限制。例如，可以使用规则将雪铲优惠限制为仅提供给居住在下雪地区的客户。
