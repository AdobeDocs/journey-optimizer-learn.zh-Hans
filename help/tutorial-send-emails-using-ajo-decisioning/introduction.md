# 使用Decisioning个性化电子邮件

在本教程中，我们将探索使用Adobe Journey Optimizer Decisioning发送个性化电子邮件优惠的实际用例。 方案很简单 — 根据客户的用户档案属性（如年收入和忠诚度状态）向客户发送有针对性的电子邮件优惠。
在本教程中，我们将逐步进行操作 — 从Adobe Experience Platform中创建用户档案，到配置电子邮件渠道，最后是通过决策构建和执行自动选择最相关选件的营销活动。

## 本教程的先决条件

* 对Experience Platform的访问权限

* 基本了解Experience Platform概念（用户档案、受众、数据集）

* 熟悉Journey Optimizer
