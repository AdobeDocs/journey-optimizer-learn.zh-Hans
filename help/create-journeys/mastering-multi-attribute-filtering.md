# 掌握用于有效数据管理的多属性过滤

使用查找架构时，通常需要根据多个属性过滤对象列表。 此视频介绍如何使用嵌套筛选条件()函数对查找架构执行多属性筛选。 当您需要根据多个条件缩小对象列表时（例如按ID和颜色过滤购物车中的产品），此功能特别有用。
