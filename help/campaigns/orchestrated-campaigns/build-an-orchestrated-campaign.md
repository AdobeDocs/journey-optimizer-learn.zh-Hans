# 构建精心策划的营销活动

了解如何通过构建结合受众、数据操作和多渠道投放的工作流，在 Adobe Journey Optimizer 中设计和执行精心策划的营销活动。本视频逐步介绍了营销活动计划、从关系型数据创建受众、数据扩充与重复数据删除，以及使用多个流来支持高级用例，如弃购跟进、流失预防和补货通知。
