# 构建编排的营销活动

了解如何通过构建结合受众、数据操作和多渠道投放的工作流，在Adobe Journey Optimizer中设计和执行编排的营销活动。 此视频介绍活动计划、从关系数据创建受众、扩充和重复数据删除，以及使用多个流支持高级用例，如放弃的购物车跟进、流失预防和股票回购通知。
