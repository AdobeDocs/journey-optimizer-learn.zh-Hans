# 创建WhatsApp历程

本教程将指导您使用Adobe Journey Optimizer创建多步WhatsApp历程。

您将学习的内容：

* 设置历程，以发送邮轮预订确认，等待三天，然后发送有针对性的追加销售消息。
* 定义受众、配置WhatsApp操作以及使用批准的模板个性化消息。
* 在发布之前使用媒体和动态变量预览和测试消息。

非常适合于希望通过WhatsApp自动实现个性化客户参与度的营销人员。
