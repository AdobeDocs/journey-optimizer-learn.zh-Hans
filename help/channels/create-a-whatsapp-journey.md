# 创建 WhatsApp 历程

本教程将指导您使用 Adobe Journey Optimizer 创建多步骤 WhatsApp 历程。

您将学习的内容：

* 设置发送邮轮预订确认的历程，等待三天，然后投放有针对性的追加销售消息。
* 定义受众、配置 WhatsApp 操作以及使用批准的模板对消息进行个性化设置。
* 在发布之前，使用媒体和动态变量预览和测试消息。

非常适合希望能自动通过 WhatsApp 执行个性化客户参与工作的营销人员。
