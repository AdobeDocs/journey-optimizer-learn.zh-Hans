# 创建 Web 推送通知

Web 推送允许网站直接向用户设备发送通知，即使浏览器处于后台也是如此。 了解如何在 Adobe Journey Optimizer 中创建和发送 Web 推送通知。本教程将指导您选择渠道配置、创作消息、选择受众和发布营销活动，以拓展除移动设备之外的更多接触方式。
