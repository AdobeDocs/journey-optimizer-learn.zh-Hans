# Web编辑器中的单页应用程序(SPA)支持

了解如何在Journey Optimizer Web编辑器中创作单页应用程序(SPA)。

此视频介绍如何：

* 使用浏览模式发现SPA视图
* 在当前视图上执行创作
* 将网站修改应用于多个视图或发现的所有视图
* 对修改执行批量操作”

欲知更多信息，请访问 [Web渠道入门文档](https://experienceleague.adobe.com/docs/journey-optimizer/using/web/get-started-web.html).
